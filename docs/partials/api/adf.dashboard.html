dashboard
service in module adf

Description

The dashboard holds all options, structures and widgets.

Methods

  • id()

    Creates an ongoing numeric id. The method is used to create ids for columns and widgets in the dashboard.

  • idEqual(id, other)

    Checks if the given ids are equal.

    Parameters
    ParamTypeDetails
    idstring

    widget or column id

    otherstring

    widget or column id

Properties

  • widgets

    Array of registered widgets.

  • widgetsPath

    Default path for widgets.

  • structures

    Array of registered structures.

  • messageTemplate

    Template for messages.

  • loadingTemplate

    Template for widget loading.

  • sets

    locale of adf.

  • hold

    all of the locale translations.

  • the

    active locale setting.

  • translation

    function passed to templates.
